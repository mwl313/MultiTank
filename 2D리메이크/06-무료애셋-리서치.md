## ⭐ 최종 추천: Kenney 조합 (Top-Down Tanks + Shooter + Tower Defense)

### 선정 이유
- **Kenney**의 모든 애셋은 동일한 깔끔한 플랫/미니멀 디자인 스타일을 공유
- 탱크(밀리터리) + 몬스터(판타지/크리쳐)라는 하이브리드 컨셉을 동일 아트 스타일로 자연스럽게 믹스 가능
- **CC0 라이선스** (완전 무료, 출처 표기 불필요)
- **게임잼에 최적** — 다운로드 즉시 사용 가능, 라이선스 고민 없음

## 실행 계획 (제안)

### Phase 1 — 애셋 다운로드 및 확인
```bash
# Kenney Top-Down Tanks
wget https://kenney.nl/media/pages/assets/top-down-tanks/.../kenney_top-down-tanks.zip

# Kenney Top-Down Shooter
wget https://kenney.nl/media/pages/assets/top-down-shooter/.../kenney_top-down-shooter.zip

# Kenney Tower Defense (Top-Down)
wget https://www.kenney.nl/media/pages/assets/tower-defense-top-down/.../kenney_tower-defense-top-down.zip
```

### Phase 2 — 프로젝트 assets 폴더에 정리
```
assets/
  tanks/       ← Kenney Top-Down Tanks
  enemies/     ← Kenney Top-Down Shooter (좀비/인간형) + Tower Defense (몬스터)
  tiles/       ← Kenney Tower Defense 타일 + Top-Down Shooter 타일
  effects/     ← Kenney Top-Down Tanks (폭발/포탄)
  ui/          ← 추후 별도 제작 또는 추가 팩
```

### Phase 3 — 게임 엔진 적용
- config에 스프라이트 경로 및 크기 설정
- 탱크 스프라이트: 운전수/포수 시점의 회전 처리
- 적 스프라이트: 여러 종류에 각기 다른 HP/속도/패턴 연결
