# 2D 탑다운 뱀서라이크 — 차별점 아이데이션

> 작성일: 2026-07-18
> 기반: 3D FPS 협동 탱크 게임 → 2D 탑다운 리메이크
> 핵심 유지: **스플릿 컨트롤** (운전수 + 포수 분리)

---

## 코어 질문

> "수많은 뱀서라이크 게임과 뭐가 달라지지?"

일반 뱀서라이크는 **혼자서 도망다니며 적들이 자동으로 죽는 게임**.
이 게임은 **같은 탱크에 탄 두 명이 각자 다른 역할을 하며 함께 싸우는 게임**.

---

## ① 레벨업 두 갈래 — 운전수/포수 독립 빌드

기존 뱀서라이크의 3지선다를 **역할별로 분리**.

### 운전수 업그레이드 라인
| 업그레이드 | 효과 |
|:----------|:------|
| 엔진 출력 ↑ | 이동속도 증가 |
| 조향 성능 ↑ | 회전 반경 감소, 민첩성 향상 |
| 장갑 강화 | 피해 감소 (퍼센트 경감) |
| 스테미나 | 스프린트 게이지 증가 |
| 충돌 데미지 ↑ | 램 어택 피해량 증가 |

### 포수 업그레이드 라인
| 업그레이드 | 효과 |
|:----------|:------|
| 재장전 속도 ↑ | 발사 간격 감소 |
| 탄종 변경 | 산탄/관통/유도탄/지뢰/확산탄 |
| 폭발 범위 ↑ | 스플래시 범위 증가 |
| 탄창 용량 ↑ | 연사 가능 횟수 증가 |
| 치명타 확률/데미지 | 헤드샷/취약부 명중 보너스 |

### 핵심 규칙
- 플레이어 수에 관계없이 레벨업 UI는 **운전수/포수가 각각 분리되어 표시**
- 2인 플레이: 각자 자신의 역할 업그레이드만 선택 가능
- 솔로 플레이 (AI 동승 시): 플레이어가 선택한 역할 업그레이드 + AI가 나머지 역할 자동 선택
- 리플레이밸류: 같은 탱크여도 빌드 방향이 매번 달라짐

---

## ② 운전수도 어택 — "운전수는 전사다"

운전수는 단순히 도망만 다니지 않음. **탱크 자체가 무기**.

### 운전수 액션

| 액션 | 조작 | 효과 |
|:----|:----|:------|
| **드리프트 슬라이딩** | 방향키 + 액션키 | 옆구리로 적 쓸기, 넉백 + 데미지 |
| **스매시** | 전방 가속 + 액션키 | 적을 벽/장애물로 밀어넣기, 추가 데미지 + 기절 |
| **회피 드라이브** | 더블탭 방향키 | 순간 가속, 무적 프레임, 적 사이 돌파 |
| **백점프** | 후진 + 액션키 | 순간 후퇴 + 먼지 퍼트려 시야 방해 |

### 전략적 의미
- **포수는 정밀 조준/화력 집중** → 탱크 멈추거나 속도 줄여야 명중률↑
- **운전수는 포수 사각 커버 + 적 정리** → 움직이면서 램 어택으로 경량 적 처리
- 운전수 혼자서도 약한 적은 잡을 수 있어야 함 (헛심 안 쓰게)

---

## ③ 전장 이동이 게임플레이다

기존 뱀서라이크: **한 맵에서 끝까지 버티기** (정적).
이 게임: **탱크가 전장을 누비며 여러 미션 수행** (동적).

### 미션 타입

| 미션 | 설명 | 게임플레이 임팩트 |
|:----|:------|:-----------------|
| **후퇴전** | 뒤에서 적이 밀려오며 앞으로 계속 이동 | 화면이 스크롤, 멈추면 위험 |
| **거점 방어** | 특정 지점에서 시간 동안 버티기 | 포수 집중 사격 + 운전수 위치 선정 |
| **돌파** | 적진 뚫고 반대편까지 이동 | 최단 경로 vs 안전 경로 선택 |
| **호위** | NPC 호송차량 보호하며 이동 | 호위 대상 기동성에 맞춰야 함 |
| **섬멸** | 특정 지역 내 모든 적 제거 | 클리어 조건이 시간이 아니라 전멸 |
| **탈출** | 카운트다운 동안 필드 끝까지 도망 | 스피드가 생명 |

### 맵 디자인 원칙
- 각 미션마다 맵 형태가 다름 (좁은 골목/넓은 평원/장애물 밀집)
- 이동 간에 맵이 전환되는 느낌 (페이드 인/아웃 or 자연스러운 스크롤)
- 파괴 가능한 장애물, 엄폐물, 고저차(2D 탑다운이라도 일부 블록 처리)
